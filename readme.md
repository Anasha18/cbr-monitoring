# Информация по проекту.

# Функциональные требования:

* пользователь может смотреть курс популярных валют с [cbr-api](https://www.cbr-xml-daily.ru/daily_json.js).

* пользователь может подписаться/отписаться рассылки курса
  определенной валюты по своему уникальному `telegram_id` (каждые 6 часов).

* конвертер валют (откуда/куда и сколько).

* система хранит историю курсов для каждой
  отслеживаемой валюты.

# Нефункциональные требования:

* стек: Java, Docker, Hibernate, Postgresql,
  TelegramBot.
